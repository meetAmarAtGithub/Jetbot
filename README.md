## Instructions for running this program
### There are 4 .ipython files in our project
### 1. lanedetection.ipython :
####   This program is responsible for detecting lanes. 
####   Input : image. 
####   Output : image with detected lanes.
### 2.  autobot_custom1.ipnyb :
####  This program is used in training the model with input dataset.
####  vgg16 is used for transfer learning. Hot encoding is used for classifying all labels using single CNN. Hence, only single weight and structure file is output.dcte
### 3.  main_image.ipnyb:
####  Input : Single image frame and pre-traned structure and weight of the model.
####  Outout : Detected lanes and recognized object.
### 4.  main_jetbot.ipnyb:
#### Input  : Single image frame and pre-traned structure and weight of the model.
####  Outout : Robot movements.
